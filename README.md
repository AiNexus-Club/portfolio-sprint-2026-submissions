## 📖 About

Portfolio Sprint 2026 was a 4-hour frontend development sprint organized by the **AI Nexus Club** for students of the **Department of Artificial Intelligence & Machine Learning, Acropolis Institute of Technology & Research, Indore**.

The objective of the event was to encourage students to design, build, deploy, and showcase a professional portfolio website under a strict time limit while following industry-standard development practices.

---

## 🎯 Objectives

- Build a professional portfolio website from scratch.
- Learn Git & GitHub workflow.
- Deploy projects on modern hosting platforms.
- Practice clean UI/UX design.
- Showcase projects, skills, and achievements.
- Simulate a real-world frontend development sprint.

## 📝 Event Rules

Participants were required to:

- Build a unique professional portfolio.
- Create a **public GitHub repository**.
- Make **minimum 5 meaningful commits**.
- Deploy the portfolio before the deadline.
- Include:
  - Professional Introduction
  - Skills
  - Projects
  - Education
  - Achievements
  - Contact Information

Pre-built portfolios or copied work were not allowed.
