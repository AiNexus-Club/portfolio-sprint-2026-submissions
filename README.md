## 📖 About

**Portfolio Sprint 2026** was a 4-hour frontend development sprint organized by the **AI Nexus Club** for the students of the **Department of Artificial Intelligence & Machine Learning** at **Acropolis Institute of Technology & Research, Indore**.

The event challenged participants to design, build, deploy, and publish a professional portfolio website within a strict four-hour time limit while following modern development workflows and GitHub best practices.

The objective was not only to improve frontend development skills but also to help students build a portfolio that can be used for internships, placements, hackathons, and professional networking.

---

## 🎯 Objectives

* Build a professional portfolio website from scratch.
* Learn industry-standard Git & GitHub workflow.
* Practice responsive web design.
* Deploy projects to modern hosting platforms.
* Showcase technical skills, projects, and achievements.
* Encourage creativity while maintaining professional standards.

## 📌 Rules & Requirements

Participants were required to:

* Create a unique portfolio website.
* Maintain a **public GitHub repository**.
* Make **at least 5 meaningful commits**.
* Deploy the website before the submission deadline.
* Include:

  * About
  * Skills
  * Projects
  * Education
  * Achievements
  * Contact Information

Pre-built templates, copied portfolios, and single-commit submissions were discouraged.
